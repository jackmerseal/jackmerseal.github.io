CRUD Operations on Sports Management Products

About this project:

This is a sports product management system, and was built using HTML, CSS, C#, SQL, ASP.NET Core & EntityFrameworkCore.

Problem solved:

Users can register for an account and sign themselves in, and user information is stored in a cookie once logged in. Users with the admin role can perform CRUD (Create, Read, Update, Delete) operations on different products within the SportsPro database.

Technologies used:
  • HTML
  • CSS
  • C#
  • SQL
  • ASP.NET Core
  • EntityFrameworkCore
