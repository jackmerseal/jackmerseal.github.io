CRUD Operations on Six Flags Reviews

About this project:

This project is a review application that combines some aspects of Yelp & Google. Users can leave reviews much like on Yelp, and they can also give a rating of 1-5 stars much like Google. An average rating is also calculated based on the number of stars a user selects, as well as the number of times a star button is clicked. This project was built using C# Windows Forms.

Problem solved:

Users can perform CRUD (Create, Read, Update, Delete) operations on reviews about Six Flags St. Louis, as well as give suggestions for how the park can improve.

Technologies used:
  • C#
  • Windows Forms
  • SQL
